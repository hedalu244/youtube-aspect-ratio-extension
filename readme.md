# YouTube アスペクト比拡張機能

YouTubeの動画をカスタムアスペクト比で視聴できるブラウザ拡張機能です。

## 機能

- **アスペクト比の自由な変換** - 4:3、16:9、またはカスタム比率に動画を変換表示できます。
- **スケーリング調整** - 動画の表示サイズを柔軟に選択できます
- **動画ごとの設定保存** - 気に入った設定を動画に紐づけて保存できるので、次に同じ動画を見る際に同じ設定が自動適用されます

## ユースケース

- ワイドスクリーンで、左右に黒帯が表示されてしまうときに、横に引き延ばして隠したい。
- 16:9に引き延ばされたVHSの動画を、コンテンツ本来の4:3に戻して視聴したい。
- 動画をディスプレイ全体を覆うように拡大したい。

## インストール方法

1. このリポジトリをクローンします
2. ブラウザの拡張機能ページを開きます
3. 「デベロッパーモード」を有効にします
4. 「パッケージ化されていない拡張機能を読み込む」を選択
5. クローンしたフォルダを選択します

## 使い方

1. [YouTube](https://youtube.com) を開き、適当な動画ページに移動します
2. 拡張機能のアイコンをクリック
3. 希望する設定を選択

### enable

拡張機能を有効化します。Youtube以外では常にオフの状態に見えます。万が一、Youtube上でオフになってしまうときはしばらく待って再読み込みしてください。

### Source Ratio

変換前のアスペクト比を設定します。

* **Detected** - 動画のアスペクト比を自動で検出します。
* **4:3** - 動画のうち4:3の領域を対象とします。
* **16:9** - 動画のうち16:9の領域を対象とします。
* **Custom** - 手動で数値を入力して設定します。

動画全体を対象とする場合は Detected を使えばで問題ありません。上下または左右に帯がついた状態でアップロードされている動画などでは他の選択肢を使用してください。

### Target Ratio

変換後のアスペクト比を設定します。

* **Original** - 縦横比を元の動画のまま維持します。
* **4:3** - Source Ratio で対象とした範囲が 4:3 に変換されます。
* **16:9** - Source Ratio で対象とした範囲が 16:9 に変換されます。
* **Custom** - 手動で数値を入力して設定します。

### Scaling Mode

表示する動画の大きさを設定します。

* **Show all** - Source Ratio で対象とした範囲全体が見えるように表示します。
* **Cover all** - Source Ratio で対象とした範囲が表示コンテナ全体を覆うように表示します。特に全画面表示ではディスプレイ全体が覆われます。
* **Manual** - 手動でサイズを設定します。

### Remember settings for this video

設定を動画に紐づけて記憶します。ほかの動画ページで設定を変更したとしても、設定が記憶された動画を再び見る時には記憶された設定で視聴することができます。デフォルトはオフになっています。
